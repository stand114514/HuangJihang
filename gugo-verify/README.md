## 孤狗验证

一个原生html实现可自定义的仿谷歌验证页面（组件？

## 实现功能

- 假的加载动画
- 空白处关闭验证层
- 图片选择和取消选择
- 验证错误提示
- 成功动画
- 修改各种信息（验证文字、图片）
